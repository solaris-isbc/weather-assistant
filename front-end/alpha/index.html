Weather Assistant (DEMO)

Wetterdaten von weatherbit.io

Geben Sie einfach eine Frage zum Wetter ein (auf Deutsch):


Das System kann folgende Fragen beantworten:

  • Wetter allgemein
  • Regen
  • Schnee
  • Sonne
  • Luftdruck
  • Nebel
  • Temperatur
  • Mindesttemperatur
  • Maximaltemperatur
  • Durchschnittstemperatur
  • Warme Temperaturen
  • Kalte Temperaturen
  • Sturm/Unwetter
  • Wind
  • Wolken
  • Windrichtung

... natürlich kann zusätzlich auch eine Zeit (für Uhrzeiten in den nächsten 47 Stunden und für die nächsten 15 Tage) angegeben werden, ebenso wie der Name einer Stadt.
