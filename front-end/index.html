Weather-Assistant

(BETA)

Was kann das System?

Der Weather-Assistant ist in der Lage, Fragen zum Wettergeschehen in den kommenden Tagen zu beantworten. Die Daten, die der Assistent zur Beantwortung von Queries verwendet, werden von der Weatherbit-API bereitgestellt. Diese bietet Wetterdaten für die nächsten 14 Tage, wobei es nur für die nächsten 48 Stunden Wetterdaten zu den einzelnen Stunden gibt. Sobald Sie eine Frage an das System gestellt haben, wird diese zusammen mit der Antwort des Systems anonymisiert in einer Datenbank gespeichert. Wir würden uns darüber freuen, wenn Ihr uns helfen könntet, das System zu verbessern!
Das System kann folgende Fragen beantworten:
  • Wetter allgemein
  • Regen
  • Schnee
  • Sonne
  • Luftdruck
  • Nebel
  • Temperatur
  • Mindesttemperatur
  • Maximaltemperatur
  • Durchschnittstemperatur
  • Warme Temperaturen
  • Kalte Temperaturen
  • Sturm/Unwetter
  • Wind
  • Wolken
  • Windrichtung
Des Weiteren kann das System Städtenamen und Uhrzeiten erkennen. Wird kein Stadtname genannt in der Query, dann geht das System von Regensburg aus, da dort der Server stationiert ist, an den die Anfragen gesendet werden.
